Publicationtoolitalynorth ​

Italy North CCR — CCR_* for day-ahead, CCR_id* for intraday. Smaller surface than Core (CNECs, forecasts, TTC/NTC, allocation constraints).

Italy North day-ahead final TTC/NTC ​

GET /publicationtool/ibwt/data/CCR_FinalTtcNtc

Italy North day-ahead final TTC/NTC​

GET
/publicationtool/ibwt/data/CCR_FinalTtcNtc

Final TTC and NTC values published for the Italy North day-ahead window.

Anonymous bearer token extracted from the SPA at https://publicationtool.jao.eu/core/. Role is Guest; the API is effectively public.
