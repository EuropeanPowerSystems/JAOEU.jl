Publicationtoolcoreidcca ​

Core CCR intraday continuous clearing (window A). Endpoints carry the variant prefix IDCCa_; the monitoring system endpoint is shared with Core (see PublicationToolCoreDA tag).

Allocation constraints ​

GET /publicationtool/coreID/data/IDCCa_allocationConstraint

Allocation constraints​

GET
/publicationtool/coreID/data/IDCCa_allocationConstraint

Core CCR intraday continuous clearing (window A) — Allocation constraints.

Authorizations​

PublicationToolBearer

Anonymous bearer token extracted from the SPA at https://publicationtool.jao.eu/core/. Role is Guest; the API is effectively public.

Type
HTTP (bearer)

Parameters​

Query Parameters

FromUtc*

Inclusive start of the requested window, ISO 8601 UTC with millisecond precision (yyyy-MM-ddTHH:mm:ss.000Z). Practical limit: two days per call; DST boundaries may force single-day chunks.

Type
string
Required
Format
"date-time"
ToUtc*

Exclusive end of the requested window, ISO 8601 UTC with millisecond precision (yyyy-MM-ddTHH:mm:ss.000Z).

Type
string
